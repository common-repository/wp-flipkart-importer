== Description ==

WP Flipkart Importer WordPress Plugin Import product using flipkart API .


= Features =
1. Import Product from Flipkart.
2. Custom import according to your parameter.
3. Makes autoblogging.
4. Support WPMU.
5. Custom template option.
6. Set cron option.

More detail : http://socialcms.wordpress.com/


== Installation ==


1. Upload the **wp-flipkart-importer** folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go **Flipkart Importer** in admin menu,set parameter click update button.
4. After that click  Import Product Now button.

== Frequently Asked Questions ==

* **Where to get Fk-Affiliate-Id?**

 go to https://affiliate.flipkart.com/api/api-token  

* **Where to get Fk-Affiliate-Token?**

 go to https://affiliate.flipkart.com/api/api-token  

* **How to display setting change**

 In WP Flipkart Importer see in **Display Template**.

* **How to stop auto fetch (cron)?**

 Feed Status set to inactive	 

* **How to Set default post status?**

 new post status set to you want

* **How to Add some addtinal default content?**

  Display Template section add your default content
